## Getting Started

Install the following to play with the program

### Prerequisites

Install <i>dependencies</i>.
  ```sh
  npm install
  ```



### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run
    ```
    node index.js
    ```
4. Run mysql -u root -p and upload the db and seed.
